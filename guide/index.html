# 介绍

该项目旨在打造一个启动项目，把自己从繁杂的项目配置中解救出来，另外作为巩固和总结开发工作中遇到问题的一个平台交流学习。

  • 安装 / Installation
  • 开发环境 / Develop
  • 贡献 / Contribution

# 安装 / Installation

# 克隆项目
git clone https://github.com/vue-run/run-element-admin.git

# 进入项目目录
cd run-element-adminn

# 安装依赖
npm install

# 建议不要用 cnpm 安装 会有各种诡异的bug 可以通过如下操作解决 npm 下载速度慢的问题
npm install --registry=https://registry.npm.taobao.org

# 本地开发 启动项目
npm start
